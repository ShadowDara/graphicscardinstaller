## Develep

**Install Requirements**
```sh
pip install -r requirements.txt
```

**Start the Programm**
```sh
python3 -m main
```

**Save the requirements**
```sh
pip freeze > requirements.txt
```

**Build the Application**
```sh
./build.sh
```
